OPL Game Compatibility List for USB Devices

This webpage is under development, please check back for updates. Refer to this main Github repository for more info. For owners who uses HDD/SMB as their game storage, refer instead to other sources.

These listing includes Official, UNOFFICIAL or Hacks, DEMO, SOFTWARE, ALPHA or PROTOTYPE release titles. ALPHA or PROTOTYPE releases usually includes Cheat and Developer in-game menu for debugging and testing purpose. We also prioritizing Japanese-only and not-so-usual release titles as most western titles can be found elsewhere in listing sites. This Game ID Prefixes may help you in identifying the region (release country) of the game.
